# GraphQL API Server with MySQL Database

## Installation

    ```sh
    npm install
    ```

## Usage

### Start Dev Server

    ```sh
    npm start
    ```

### Database ORM CLI

    ```sh
    npm run orm
    ```

### Synchonize database schemas with configured MySQL database and fill with dummy data

    ```sh
    npm run db:seed
    ```

### Drop Database

    ```sh
    npm run db:drop
    ```
